🚀 如何启动应用

  1. 确保已安装Node.js (>= 16.0.0)
  2. 在项目根目录运行：
    npm install
  3. 启动开发服务器：
    npm run dev
  4. 访问 http://localhost:3000
  5. 应用将直接显示主界面，无需登录

⚠️ 注意事项

  • 权限系统保留：虽然移除了登录，但权限控制组件仍然存在
  • 默认管理员权限：当前用户具有所有权限 (permissions: ['*'])
  • 登出功能禁用：点击登出不会有任何效果
  • 路由保护移除：所有页面都可以直接访问
  • 后续扩展：如需重新启用登录，只需恢复相关代码
